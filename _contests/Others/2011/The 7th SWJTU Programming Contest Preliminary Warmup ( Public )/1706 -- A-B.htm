A-B

Time Limit:1000MS  Memory Limit:65536K
Total Submit:389 Accepted:60

Description

Give two integers A and B, please calculate A-B. Note that A, B is no more than 500 digits.

Input

Each line contains two integers A and B.

Output

For each case, output a line represents the value A-B, assume that +1=1.

Sample Input 

+1 2
-1 -1
-0 -1

Sample Output 

-1
0
1
