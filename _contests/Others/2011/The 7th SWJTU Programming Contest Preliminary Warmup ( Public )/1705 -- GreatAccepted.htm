Description

7th Preliminary Warmup Problem Set


I always believe that GreatAccepted (Hzwu¡¯s POJ ID) is a boring man. He always submits a set of accepted codes of the same problem on different online judges. For example, POJ 2418 and SOJ 1704 is the same problem, with the same description and the same algorithm. After solving POJ 2418, he submitted the accepted codes to SOJ 1704 and got the problem accepted again. Actually it is meaningless to do so and GreatAccepted starts to give it up. What is more, he takes it just for checking correctness of the problem by adding new & valuable problems on SOJ.
Well, GreatAccepted knows that not only he does that, but also some others do this. There, he comes up with a terrible problem. How many problems does he really have solved before? He wants you to help him figure out the number of different problems he has solved actually. (We know the problems on the same online judge are distinct. And GreatAccepted would submit a set of accepted codes of the same problem on 3 different online judges at most)

Output

For each case, you should print a line containing the case number (beginning with 1) and output two separated integers in a line, indicating the least number of different problems GreatAccepted has solved and the most number of different problems GreatAccepted has solved if possible.
